# 06 — Guided Questions

Questions are conditional by file type.

## Universal
- What is this file?
- Does it show/reference FATLETIC?
- Is the mark clearly visible?
- Is it connected to a real product?
- Is it connected to an offer, sale, shipment, customer, or public promotion?
- What real-world date applies, and what supports it?
- Which other file supports this answer?
- Include in trademark package?

## Images
Ask who/what is shown, whether item was sold/gifted/sample, date, public posting, platform, matching invoice/order/message/shipment, original vs screenshot.

## Invoices/orders
Ask type, order number, date, products, quantity, purpose, matching images/customers/posts.

## Design files
Ask version, whether printed/sold, matching exports/products/orders, brand-history-only vs commerce relevance.

## Videos
No AI. Ask user what is shown, relevant timestamps, event type, date basis, and linked evidence.

Every answer stores value, source, confidence, and optional note.
