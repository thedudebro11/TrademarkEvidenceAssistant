# 08 — Trademark Usefulness Scoring

Organizational aid only, not a legal conclusion.

Output: 0–100, Strong/Moderate/Weak/None/Undetermined, explanation, missing elements, supporting links.

Positive factors: visible mark, real product, independent date source, order/invoice, shipment, customer, public promotion, continuous use, corroboration.

Negative factors: no visible mark, no commerce link, unknown date, only filesystem time, duplicate without added context, conceptual-only design, unsupported event.

Formula must be documented; no hidden AI; user override requires a note; never label an item legally sufficient.
