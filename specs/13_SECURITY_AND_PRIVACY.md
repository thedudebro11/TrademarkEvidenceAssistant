# 13 — Security and Privacy

Local-first. No uploads, no external APIs, no evidence analytics.

Validate paths, block traversal, never execute evidence, generate previews safely, sanitize export filenames, never overwrite exports silently.

Warn before external export because evidence may contain customer names, addresses, messages, invoices, shipping data, photos, and private records.

Evidence, databases, exports, previews, and binders remain ignored by Git.
