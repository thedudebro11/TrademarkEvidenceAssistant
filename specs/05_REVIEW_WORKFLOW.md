# 05 — Review Workflow

The app opens into the Review Queue.

Layout:
- large preview
- known metadata
- guided questions
- existing connections
- usefulness result
- decision actions

Flow:
1. load next unreviewed item
2. preview
3. show deterministic metadata
4. show duplicate/related candidates
5. ask relevant questions
6. autosave
7. connect related items
8. calculate transparent score
9. user chooses Include / Maybe / Follow-Up / Not Useful
10. next item

Keyboard: N next, P previous, I include, M maybe, X exclude, F follow-up, L link, S save.
