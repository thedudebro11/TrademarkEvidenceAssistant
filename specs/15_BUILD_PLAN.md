# 15 — Build Plan

Phase 0 Analysis: inspect repo/evidence, write plans, no app code.
Phase 1 Foundation: backend, frontend, SQLite, migrations, config, tests.
Phase 2 Scanner: hashing, metadata, duplicates, IDs, scan UI.
Phase 3 Review Queue: preview, one-item workflow, autosave, statuses.
Phase 4 Conditional Questions: file-role logic, confidence/source fields.
Phase 5 Connections: link evidence and show chains.
Phase 6 Scoring: deterministic score, explanation, override.
Phase 7 Export: approved-only copy, safe names, private map.
Phase 8 Binder: Markdown/HTML/CSV/JSON factual outputs.

v1 release gate: scan, review, connect, classify, export, and generate binder without altering originals.
