# 01 — Scope and Non-Goals

## v1 scope
- scan a local evidence directory
- create one Evidence Item per file
- extract deterministic metadata
- preview supported files
- review one item at a time
- ask conditional guided questions
- save answers
- connect related Evidence Items
- classify Useful / Maybe / Not Useful / Follow-Up
- calculate a transparent organizational score
- copy approved files into a clean package
- generate an evidence binder
- preserve originals unchanged

## Supported groups
Images, videos, PDFs, text/JSON/CSV/Markdown, PSD/XCF/SVG/AI metadata where practical.

## Excluded
AI, cloud, social APIs, USPTO filing, legal conclusions, collaboration, moving originals, plugin marketplace, generic platform architecture.
