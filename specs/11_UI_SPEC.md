# 11 — UI Specification

Pages: Setup, Scan Progress, Review Queue, Evidence Detail, Link Evidence, Review Summary, Trademark Evidence, Maybe Useful, Not Useful, Export Package, Binder, Settings.

Principles: preview-first, one primary decision per screen, minimal navigation, clear progress, plain language, autosave visibility, keyboard friendly.
