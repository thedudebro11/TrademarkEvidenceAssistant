# 04 — Scan and Metadata

- recursive scan
- ignore app/cache/database/export/generated folders
- SHA-256
- exact duplicate detection
- stable Evidence Item IDs
- incremental rescan
- preserve review data
- never rewrite originals

Extract deterministic metadata for images, video, PDF, design files where practical, and all basic filesystem facts.

Filesystem timestamps must be labeled as filesystem metadata, not proof of the real-world event date.
