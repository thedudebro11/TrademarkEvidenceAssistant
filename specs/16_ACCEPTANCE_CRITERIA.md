# 16 — Acceptance Criteria

Accepted when it scans real Fatletic evidence unchanged, creates one item per file, renders supported previews, shows metadata, supports one-at-a-time review, conditional questions, autosave, linking, explainable scoring, Include/Maybe/Follow-Up/Not Useful decisions, approved-only copy, byte-identical originals, accurate binder, documented local run commands, and passing tests.
