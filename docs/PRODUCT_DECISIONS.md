# Product Decisions

## Confirmed
1. Local browser app.
2. Original evidence files are never moved, renamed, overwritten, or deleted.
3. Only approved evidence is copied into an organized trademark evidence package.
4. Version 1 contains no AI.
5. Primary workflow: preview → metadata → relevant questions → connect evidence → classify usefulness → next file.
6. The product is intentionally narrow.
7. Primary v1 workspace: Fatletic.

## Non-goals
- legal advice
- USPTO filing
- likelihood-of-confusion analysis
- cloud sync
- collaboration
- mobile app
- AI classification
- automatic legal conclusions
